Trabalhando com Elementos

Semântica: Um html bem estruturado é melhor rankeado no buscador do Google, um bom CSS, scripts para manipular a página. Uma boa página bem escrita pode ser feita algumas reflexões:

  • Elementos que melhor descrevem o contéudo da página;
  • Tipo de alinhamento e posicionamento;
  • Cabeçalho e Rodapé;
  • Outras questões dependendo do projeto.

Títulos e Paragráfos

Os títulos devem ser hierárquicos, um títul <h1> é o título principal, melhor indicado de ser usado apenas 1. A númeração dos títulos vai de <h1> até <h6>. Os títulos devem ser hierárquicos, um títul <h1> é o título principal, melhor indicado de ser usado apenas 1. A númeração dos títulos vai de <h1> até <h6>.

Paragráfos ajudam a estruturar melhor o texto, e entre os títulos o HTML já faz um formatação, dando um espaço entre o título e o elementos acima e abaixo.

Uma boa prática é usar no máximo três tramanhos de títulos pode página. Se precisar de mais, melhor reestrutrar as páginas, deixando elas menos poluidas e melhores estruturadas.

Listas

Listas: podem ser ordenadas ou não ordenadas. Servem para fazer uma listagem de itens organizando o conteúdo. Para criar uma lista não ordenada usamos a tag <ul> e para uma lista ordenadas usamos a lista usamos %lt;ol>. A listagem dos elementos da lista usa-se a tag <li>.
