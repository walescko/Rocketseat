Vou escrever um texto aqui! Tomar muito cuidado para não fechar a tag no lugar errado.

Cuidar para fechar as tags na ordem hierarquica correta para evitar problemas. Tipo tem uma tag <p> e na sequencia "em", devemos fechar a <em> que é uma tag filha da tag "p".

A tag <br> é para quebrar a linha em um ponto específico da página, como aqui
que tem essa tag.

o fluxo do texto HMTL é na sequencia em que o html foi escrito.

Outra coisa que temo que cuidar é o tags aninhadas, fechando as tags abertas de dentro para fora. Exemplo: abre-se uma tag <p> e depois a tag <em>, fechamos primeiro </em> e depois </p>.

O elemento HTML <strong> dá ao texto uma forte importância, e é tipicamente mostrado em negrito, muito usado no html para mostrar para quem faz a manutenção do código que é um texto de ênfase.

<head> - configurações do documento HTML.

<body> - conteúdo da página.
